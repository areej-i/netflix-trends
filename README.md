# netflix-trends

This project analyzes the Kaggle Netflix Movies and TV Shows dataset: https://www.kaggle.com/datasets/shivamb/netflix-shows uncovering trends related to content production, genres, countries, and release patterns.
The goal is to demonstrate skills in:
- Data cleaning
- Exploratory data analysis (EDA)
- Python (Pandas, Matplotlib, NumPy)
- SQL-like grouping/filter logic
- Data visualization
- Communicating insights in a clear narrative

This project is part of my Data Analyst learning roadmap.

Tools Used:

- Python
- Jupyter Notebook
- Pandas for cleaning & manipulation
- Matplotlib / Seaborn for visualization
- NumPy
- CSV dataset from Kaggle
