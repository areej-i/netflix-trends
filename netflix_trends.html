Netflix Trends

Load and Explore Data

This section loads the dataset derived from https://www.kaggle.com/datasets/shivamb/netflix-shows and checks the basic structure, cleaning up the data to be more useful by doing the following:

  • Removing duplicate rows
  • Removing rows with empty columns
  • Converting the date added to a DateTime object for better filtering
  • Renaming listed_in to genre
  • Splitting countries, directors, casts and genres into lists for better filtering
  • Cleaning the genre list to not have movies or TV shows in the name
  • Cleaning the genre list to categorize "Anime Features" and "Anime Series" as just "Anime"
  • Dropping the date added column

import pandas as pd
df = pd.read_csv("../data/netflix_titles.csv")

import matplotlib.pyplot as plt
import seaborn as sns

# Optional: make charts look nicer
sns.set(style="whitegrid")
%matplotlib inline

df_cleaned = df.drop_duplicates()
print(f"Number of rows after removing duplicates: {df_cleaned.shape[0]}")

# Remove rows with any NaN values.
df_cleaned = df_cleaned.dropna()
print(f"Number of rows after removing NaN values: {df_cleaned.shape[0]}")

# Convert "date_added" to a datetime object.
df_cleaned['date_added'] = pd.to_datetime(df_cleaned['date_added'], errors='coerce')

# Rename "listed_in" to "genre"
df_cleaned = df_cleaned.rename(columns={'listed_in': 'genre'})
print("Renamed 'listed_in' column to 'genre'.")

# Split the country column into lists of countries.
df_cleaned['country'] = df_cleaned['country'].str.split(', ')

def clean_country_list(country_list):
    cleaned = []
    for c in country_list:
        c = c.strip()          # remove spaces
        c = c.rstrip(",")      # remove trailing comma
        if c != "":            # ignore empty entries
            cleaned.append(c)
    return cleaned

df_cleaned['country'] = df_cleaned['country'].apply(clean_country_list)

# Split the director columns into lists.
df_cleaned['director'] = df_cleaned['director'].str.split(', ')

# Split the cast column into lists of actors.
df_cleaned['cast'] = df_cleaned['cast'].str.split(', ')

# Split the genre column into lists of genres.
df_cleaned['genre'] = df_cleaned['genre'].str.split(', ')

def clean_genre_list(genre_list):
    cleaned = []
    for g in genre_list:
        g = g.replace(" Movies", "").replace(" TV Shows", "").replace("TV ", "").replace("Anime Features", "Anime").replace("Anime Series", "Anime")
        cleaned.append(g)
    return cleaned

df_cleaned['genre'] = df_cleaned['genre'].apply(clean_genre_list)

# Drop the "date_added" column.
df_cleaned = df_cleaned.drop(columns=['date_added'])
print("Dropped 'date_added' column.")

Findings
1. Geographic Content Distribution


    

  • The United States is the primary distributer for both TV shows and movies on Netflix with 2444 movies and 41 shows in the dataset, following with India at second place for film production with 932 movies and the United Kingdom at second place for TV shows with 23 shows. This data is consistent with the massive billion-dollar Hollywood and Bollywood industries which dominate other media industries globally.
    • 



2. Genre Trends and Regional Preferences


    

  • The most common genre globally is International for both television and films, reflecting the worldwide user base that primarly consume media around their respective countries. This is further shown where the most common TV genre from Japan and South East Asia was Anime and British from the United Kindom. The genre category of International also suggests a need for more granular labeling within Netflixs catalog metadata.
    • 



3. TV Show Longevity


    

  • The longest running television series include The Great British Baking Show, Call the Midwife, and Naruto, all spanning 9 seasons. The average length for a TV series on Netflix is 1.83 seasons, showing that most series available on Netflix are short-run. This may suggest Netflix originals have less user demand for more seasons or greater popularity with limited-run series. The longevity of the longest shows may reflect the popularity of British TV as well as Anime.
    • 



4. Directors and Creators


    

  • Jan Suter has directed the most movies, a total of 21 films in the dataset.
    • 



5. Temporal Trends


    

  • Movie output increased steadily from 2000 to 2017, reflecting rapid digital production and global distribution trends in the streaming era. Post-2017, the number of films begins to decline while TV shows have continued to increase each year. Some possible explanations for this is:
      

    • Netflix acquired fewer older titles after 2017.
      • 

    • A shift from film licensing to TV show and original series production.
      • 

    • Rise in demand for TV series.
      • 

    • Dataset limitations: newer titles may not yet reflect full catalog availability.
      •
