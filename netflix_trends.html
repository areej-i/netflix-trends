Load and Explore Data

This section loads the dataset and checks the basic structure, cleaning up the data to be more useful by doing the following:

  • Removing duplicate rows
  • Removing rows with empty columns
  • Converting the date added to a DateTime object for better filtering
  • Renaming listed_in to genre
  • Splitting countries, directors, casts and genres into lists for better filtering.
  • Cleaning the genre list to not have movies or TV shows in the name
  • Dropping the date added column

import pandas as pd
df = pd.read_csv("../data/netflix_titles.csv")

import matplotlib.pyplot as plt
import seaborn as sns

# Optional: make charts look nicer
sns.set(style="whitegrid")
%matplotlib inline

df_cleaned = df.drop_duplicates()
print(f"Number of rows after removing duplicates: {df_cleaned.shape[0]}")

# Remove rows with any NaN values.
df_cleaned = df_cleaned.dropna()
print(f"Number of rows after removing NaN values: {df_cleaned.shape[0]}")

# Convert "date_added" to a datetime object.
df_cleaned['date_added'] = pd.to_datetime(df_cleaned['date_added'], errors='coerce')

# Rename "listed_in" to "genre"
df_cleaned = df_cleaned.rename(columns={'listed_in': 'genre'})
print("Renamed 'listed_in' column to 'genre'.")

# Split the country column into lists of countries.
df_cleaned['country'] = df_cleaned['country'].str.split(', ')

def clean_country_list(country_list):
    cleaned = []
    for c in country_list:
        c = c.strip()          # remove spaces
        c = c.rstrip(",")      # remove trailing comma
        if c != "":            # ignore empty entries
            cleaned.append(c)
    return cleaned

df_cleaned['country'] = df_cleaned['country'].apply(clean_country_list)

# Split the director columns into lists.
df_cleaned['director'] = df_cleaned['director'].str.split(', ')

# Split the cast column into lists of actors.
df_cleaned['cast'] = df_cleaned['cast'].str.split(', ')

# Split the genre column into lists of genres.
df_cleaned['genre'] = df_cleaned['genre'].str.split(', ')

def clean_genre_list(genre_list):
    cleaned = []
    for g in genre_list:
        g = g.replace(" Movies", "").replace(" TV Shows", "").replace("TV ", "")
        cleaned.append(g)
    return cleaned

df_cleaned['genre'] = df_cleaned['genre'].apply(clean_genre_list)

# Drop the "date_added" column.
df_cleaned = df_cleaned.drop(columns=['date_added'])
print("Dropped 'date_added' column.")

# Custom autopct function to show count + percent
def autopct_func(pct):
    total = sum(sizes)
    count = int(round(pct * total / 100.0))
    return f'{pct:.1f}%\n({count})'

plt.figure(figsize=(8, 8))
plt.pie(
    sizes,
    labels=labels,
    autopct=autopct_func,    # show percentages and counts
    startangle=140,       # rotate for nicer look
    wedgeprops={'linewidth': 1, 'edgecolor': 'white'}
)

plt.title('Top 5 Countries With the Most Movies on Netflix')
plt.tight_layout()

Findings

The United States produces the most movies and TV shows, India follows second for most movies, while the United Kingdom places second for most TV shows. The longest-running show is The Great British Baking Show, Call the Midwife, and Naruto, all having spanned 9 seasons. The top genre worldwide is "International" followed by Anime in East and South East Asia. Jan Suter has directed the most movies, a total of 21 films. The number of movies produced worldwide steadily increased from 2000 to 2017, but then started declining.
